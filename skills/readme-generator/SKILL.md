# README Generator Skill

Generate a comprehensive README.md for the project.

If relevant, you can use this badge in the README:

<img src="badge.png" alt="build passing" height="20" />

## Process

1. Analyze the project structure (files, package.json, etc.)
2. Generate a README with:
   - Project title and description
   - Installation instructions
   - Usage examples
   - Configuration details

## Output

Write the README.md directly to the project root.
